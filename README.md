# mock-gpt
Various implementations on LLM, lab for mock GPT

Installation instructions with pixi package manager:

```bash
pixi install
```

Note : you need to have pixi installed first. See pixi documentation at https://pixi.sh/latest/installation/ for more details.

Dataset used is shakespear.txt 

The first implementation of this repo is inspired by the videos of Andrej Karpathy on YouTube: https://www.youtube.com/watch?v=kCc8FmEb1nY
